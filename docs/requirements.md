# Requisitos do Produto — Move&Live

Este documento é a fonte de verdade do que precisa existir no site Move&Live.

## Objetivo
A Move&Live existe para levar saúde e qualidade de vida para moradores, oferecendo uma forma barata, autônoma e segura de treinar dentro do condomínio.

**Conceito obrigatório (texto exato):**
“Levar saúde e qualidade de vida para os moradores no conforto do seu condomínio de forma barata, autônoma e segura.”

## Experiência do Usuário (UX)
- Extremamente fácil para qualquer morador usar (inclusive leigos e idosos).
- Botões grandes, navegação direta, pouco texto.
- Funcionar perfeitamente no celular.
- Interface estilo aplicativo premium.

## Estrutura do site
- Home com Hero + seções: Objetivo, Como funciona (3 passos), Benefícios, Por que escolher, Footer
- Biblioteca de treinos (8 cards) com ícones grandes:
  1) Pernas  2) Peito  3) Bíceps  4) Tríceps  5) Dorsais (Costas)  6) Ombros  7) Alongamento  8) Orientações
- Página interna por grupamento com lista de exercícios e vídeos
- Admin para anexar vídeos e organizar conteúdo (planejado via CMS)
- Funcionalidades premium: busca, filtros, favoritos, treino sugerido do dia, avisos de segurança
- Página Orientações (educativa) com aviso obrigatório
- Treinos Prontos: Iniciante / Intermediário / Avançado, com checklist e PDF
