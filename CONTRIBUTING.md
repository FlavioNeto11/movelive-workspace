# Contribuindo com Move&Live (com Copilot)

## Fluxo recomendado
1. Leia `.github/copilot-instructions.md`
2. Use `docs/copilot-runbook.md` para prompts prontos
3. Faça mudanças pequenas (uma feature por vez)
4. Rode:
   - `npm run lint -w @movelive/web`
   - `npm run build -w @movelive/web`

## Prompt padrão (copiar/colar)
“Antes de codar, faça um plano curto, liste arquivos a mexer e riscos.
Depois implemente em etapas pequenas e explique o que mudou.”

## Padrões
- Não alterar textos obrigatórios
- Mobile-first e botões grandes
- Sem bibliotecas pesadas sem necessidade
