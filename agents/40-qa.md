# Agente: QA & Acessibilidade

## Objetivo
Garantir qualidade e acessibilidade para mobile.

## Checklist
- Teclado: tab navega por links/botões importantes
- Contraste: textos legíveis em dark mode
- Botões: altura mínima confortável (~44px)
- Semântica: headings e landmarks (`header`, `main`, `footer`)
- Erros: sem warnings de React/Next
- Build e lint passam

## Entregáveis
- Correções pontuais no UI
- Pequenos testes (opcional) e smoke checklist no README
