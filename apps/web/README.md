# @movelive/web

Frontend do Move&Live (Next.js + Tailwind).

## Rodar
```bash
npm install
npm run dev -w @movelive/web
```

## Variáveis
Copie `.env.example` → `.env.local` e ajuste se quiser.

## Conteúdo
- Dados exemplo em `src/lib/content/sampleData.ts`
- Trocar por CMS: implementar `CmsProvider` e trocar em `src/lib/content/index.ts`
